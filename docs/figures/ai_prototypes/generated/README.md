# AI-generated visual prototypes

This directory stores non-authoritative raster prototypes used only for visual
composition review. These images are not paper evidence figures and are not used
directly by LaTeX. Any accepted design idea must be recreated in the editable
Mermaid, DOT, or draw.io source and regenerated through
`python scripts/render-paper-figures.py`.

- `nsi-architecture-prototype-20260523.png`: GPT-image prototype used to guide
  the lane-based redraw of `docs/figures/src/nsi_architecture.mmd`.
